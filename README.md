# mygames

一个基于 **Three.js** 的单页面 3D 交互演示项目，主题是 **T4 噬菌体感染大肠杆菌**。当前仓库以一个 `index.html` 文件承载主要功能，适合作为作品原型、教学演示或后续工程化拆分的基础。

## 项目简介

这个项目模拟了噬菌体在水体环境中搜索、接近、附着并向大肠杆菌注入遗传物质，最终引发裂解的过程。整体视觉风格偏深海、低光、科幻生物演示，支持：

- 手动操控噬菌体
- 自动演示感染流程
- 感染阶段提示
- 鼠标与触屏交互
- 基于程序化几何体的生物模型
- 可选 GLTF 模型替换

## 当前仓库结构

目前仓库结构非常精简：

```text
.
├── README.md
└── index.html
```

其中：

- `README.md`：项目说明文档
- `index.html`：页面结构、样式、Three.js 场景、交互逻辑、音效与主循环的统一入口

## 技术栈

- **HTML / CSS / JavaScript**
- **Three.js**（通过 CDN `importmap` 引入）
- **Web Audio API**（用于简单音效）
- **GLTFLoader**（用于尝试加载可选外部模型）

当前项目没有引入打包器、测试框架或前端工程化目录，属于浏览器直接运行的静态原型。

## 运行方式

### 方式 1：直接在浏览器中打开

如果浏览器允许本地模块与资源加载，可以直接打开 `index.html`。

### 方式 2：推荐使用本地静态服务器

更推荐通过本地 HTTP 服务启动，避免某些浏览器对模块、资源或本地文件访问的限制。

例如：

```bash
python3 -m http.server 8000
```

然后在浏览器中访问：

```text
http://localhost:8000
```

## 游玩 / 交互说明

进入页面后，用户需要先选择：

- **不适等级**：0～3 级
- **控制模式**：
  - `手动操控`
  - `自动演示`

### 鼠标操作

- **左键拖拽**：手动操控噬菌体
- **中键拖拽**：旋转视角
- **滚轮滚动**：缩放视角

### 触屏操作

- **单指滑动**：旋转视角
- **双击后拖动**：操控噬菌体
- **双指操作**：旋转 + 缩放

### 其他

- 点击右上角按钮可暂停
- 按 `P` 可切换暂停状态

## 核心玩法 / 演示流程

项目围绕一个感染状态机运行，主要阶段包括：

1. **search**：搜索宿主
2. **approach**：接近宿主
3. **attach**：尾丝附着
4. **inject**：DNA 注入
5. **infected**：感染完成
6. **lysis**：细胞裂解

这些阶段会驱动：

- 文本提示变化
- 噬菌体朝向变化
- DNA 缩放与注入表现
- 菌体发光增强
- 裂解粒子爆发效果

## 项目架构说明

虽然当前只有一个文件，但逻辑上可以分成以下几个层次：

### 1. UI 与页面层

负责：

- 开场选择遮罩
- HUD 操作提示
- 感染阶段说明
- 暂停菜单
- 单局完成菜单

### 2. 背景氛围层

使用独立的 2D Canvas 绘制背景粒子，制造“深海浮游”效果。

### 3. 3D 场景层

在 Three.js 中初始化：

- Renderer
- Scene
- Camera
- Lights
- 球形活动边界

### 4. 生物实体层

包含两个主要实体：

- **T4 噬菌体**
  - 头部
  - DNA
  - 颈部
  - 尾鞘
  - 基板
  - 尾丝

- **大肠杆菌**
  - 胶囊形主体
  - 发光轮廓
  - 菌毛
  - 鞭毛

### 5. 输入控制层

统一处理：

- 鼠标拖拽
- 中键旋转视角
- 滚轮缩放
- 触控手势

### 6. 状态机与动画层

通过主循环逐帧推进感染阶段，并同步更新：

- 两个实体的位置与速度
- 相机跟随
- 视觉特效
- UI 状态
- 裂解动画

## 模型与资源说明

当前项目默认使用**程序化建模**创建噬菌体和大肠杆菌。

同时，代码中预留了对以下可选模型文件的尝试加载：

- `E.coli.glb`
- `full_bac.glb`

如果这些文件存在，则会替换默认的程序化模型；如果不存在，则自动回退到内置几何体版本。

> 注意：仓库当前未包含上述 `.glb` 文件。

## 适合新人的阅读顺序

如果你是第一次接触这个仓库，建议按下面顺序阅读代码：

1. `index.html` 中的页面结构和启动入口
2. 模式选择与 `initGame(...)`
3. 场景初始化（renderer / scene / camera / lights）
4. 大肠杆菌与噬菌体模型构建
5. `infection` 状态对象与阶段切换函数
6. 鼠标 / 触摸输入处理
7. `animate()` 主循环

这样更容易先建立“全局地图”，再深入具体动画细节。
